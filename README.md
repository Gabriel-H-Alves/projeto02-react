# ⚛️ Lista de Exercícios React — Arrays

> Conjunto de exercícios práticos focados no uso de arrays em React, explorando manipulação e renderização de dados no JSX.

---

## 📌 Sobre o Projeto

Este projeto reúne uma lista de exercícios desenvolvidos com React, com foco na utilização de **arrays dentro de componentes funcionais**.

As atividades demonstram como acessar, manipular e exibir dados armazenados em arrays diretamente no JSX, utilizando expressões JavaScript.

---

## 🎓 Contexto Acadêmico

Projeto desenvolvido na faculdade **Flamingo**, durante o curso de **Análise e Desenvolvimento de Sistemas (ADS)**, como parte da segunda lista de exercícios em React.

---

## ⚙️ Tecnologias Utilizadas

* React
* JavaScript (ES6+)
* JSX

---

## 🧠 Conceitos Praticados

Durante os exercícios, foram aplicados conceitos fundamentais como:

* Criação de arrays
* Acesso a elementos por índice (`array[index]`)
* Operações com valores de arrays
* Propriedade `.length`
* Interpolação de dados no JSX
* Renderização dinâmica de informações

---

## 🖥️ Estrutura do Código

O projeto é baseado em um único componente (`App.jsx`), onde os exercícios são organizados de forma sequencial.

Cada exercício explora um cenário diferente, como:

* Exibição de itens específicos de um array
* Operações matemáticas com valores armazenados
* Construção de frases dinâmicas
* Contagem de elementos
* Simulação de listas (tarefas, filmes, linguagens, etc.)
